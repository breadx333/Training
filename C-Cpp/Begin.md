Begin1°. Дана сторона квадрата a. Найти его периметр P = 4·a. 

```
#include <stdio.h>

int main()
{
    int P, a;

    printf("Enter <a>: ");
    scanf("%d", &a);

    P = 4 * a;

    printf("Perimeter = %d\n", P);

    return 0;
}
```
